    How to Use

    1. Create a path in Google Earth:
      • Open Google Earth Pro (desktop) or Google Earth Web
      • Use the "Add Path" or "Add Line" tool to draw your route
      • Save/Export the path as a KML file
    2. Upload the KML file using the form above
    3. Configure options:
      • Interval: Set the distance between markers (e.g., 60 meters)
      • Offset: Use the slider to shift markers perpendicular to the path. Positive values move to the right, negative to the left (as you face the direction of the path). Use this to place markers on the sidewalk.
      • Label & Color: Customize marker appearance
    4. Click "Generate KML" to download the output file
    5. Import into Google Earth: File → Import → select the downloaded KML

    About Offset

    The offset shifts markers perpendicular to the path direction. This is useful when your path follows the center of a road and you want markers on the sidewalk.

    Example: If your path is along the road center and you want markers on the right sidewalk, use a positive offset equal to half the road width plus the distance to the sidewalk.

    Interactive Preview: The map shows both the original path (blue line) and the marker positions (red dots). Drag the offset slider to see markers move in real-time. You can also drag individual markers to fine-tune their position if needed.
